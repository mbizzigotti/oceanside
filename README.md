# Oceanside

WORK IN PROGRESS

# Setup

```sh
clang build.c -o build
```

> [!NOTE]
> Building on Windows requires **clang** to be installed;
> the easiest way to install clang is from [github.com/llvm/releases/latest](https://github.com/llvm/llvm-project/releases/latest) as `LLVM-X.Y.Z-win64.exe`

# Build
```sh
./build
```
